This project uses a dataset named 'globalterrorismdb_0718dist.csv' which needs to be downloaded from "https://www.kaggle.com/datasets/START-UMD/gtd/data"

The code file which is named 'Design 1 code.ipynb' is written in Python 3 and can be executed either in Jupyter Notebook environment or Visual Studio Code. The Seaborn library is utilized for data visualization.

To run the code, ensure you have Python 3 installed on your system along with the necessary libraries. You can install the required libraries using pip: 
pip install seaborn pandas matplotlib
