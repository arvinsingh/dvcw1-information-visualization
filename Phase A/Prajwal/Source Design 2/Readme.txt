This Folder contains a dataset named 'globalterrorismdb_0718dist.csv' which is used in this project. 

The code which is named 'design_2_ code.ipynb' is written in Python 3 and can be executed either in Jupyter Notebook environment or Visual Studio Code. The seaborne library is utilized for data visualization.

To run the code, ensure you have Python 3 installed on your system along with the necessary libraries. One can use "pip install seaborn" to download the necessary packages
